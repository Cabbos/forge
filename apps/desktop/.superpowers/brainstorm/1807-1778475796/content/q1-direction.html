前端重设计 — 方向选择

当前痛点：侧边栏 + 三栏布局占了太多空间，chat 区域被挤压

当前架构

Sidebar (240px)Session 列表 + New Session 按钮
Chat (1fr)MessageList + InputBar
HubPanel (320px)Skills / MCP / Hooks 管理

你最想改什么？

A

简化布局

Sidebar 缩到最小（仅 session 切换图标），HubPanel 可收起，Chat 占满屏幕。像 VS Code 的禅模式。

B

Claude Code TUI 风格

终端感设计，等宽字体为主，极简 chrome，全键盘操作。消息流密集紧凑。

C

现代 AI 工作台

保留三栏但重新设计比例和视觉。参考 Claude.ai / ChatGPT 的干净卡片风格。更圆润、更多留白。

D

先不动布局，改细节

修颜色、间距、字体、动画过渡。当前布局能用，先打磨到好看。
