关键决策：HubPanel 的默认状态是什么？

A

默认隐藏，需要时滑出

HubPanel 默认收起（只留一个小图标），需要看 Skills/MCP 时从右侧滑出覆盖。Chat 始终全宽。参考 Arc Browser 侧边栏。

B

始终可见，更窄

HubPanel 缩到 260px，始终显示。适合你经常开关 Skills 的使用习惯。Chat 区域会略窄。

C

底部面板替代侧面板

去掉右侧 HubPanel，Skills/MCP 改为底部 Tab 栏或 ⌘K 命令面板打开。Chat 获得最大空间。类似 VS Code 终端面板。
